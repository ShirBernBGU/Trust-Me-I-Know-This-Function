def fisher_yates_shuffle_recursive(arr, n=None):
    if n is None:
        n = len(arr)
    if n <= 1:
        return arr
    
    import random
    i = random.randint(0, n - 1)
    arr[n - 1], arr[i] = arr[i], arr[n - 1]
    fisher_yates_shuffle_recursive(arr, n - 1)
    return arr

This code implements the Fisher-Yates shuffle algorithm recursively to randomly shuffle an array.

The output will be **a randomly shuffled version** of the list `[1, 2, 3, 4, 5, 6, 7, 8]`.

Since it uses `random.randint()`, the exact output will be different each time you run it. For example, you might get:
- `[3, 1, 7, 8, 5, 2, 4, 6]`
- `[8, 4, 2, 6, 1, 7, 5, 3]`
- `[5, 2, 8, 1, 4, 3, 6, 7]`
- etc.

**How it works:**
1. The function starts with `n = 8` (length of the array)
2. In each recursive call:
   - It randomly selects an index `i` from 0 to `n-1`
   - Swaps the element at index `i` with the element at position `n-1` (the last unprocessed element)
   - Recursively calls itself with `n-1` to process the remaining unshuffled portion
3. The recursion stops when `n <= 1`

The result is that each element has an equal probability of ending up in any position, creating a uniformly random permutation of the original list.

**Note:** The function modifies the array in-place and also returns it, so `result` will contain the shuffled array, and the original list passed to the function will also be modified.
