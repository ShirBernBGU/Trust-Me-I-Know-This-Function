import math, cmath

def ifft(a):
    """
    Compute the inverse Fast Fourier Transform (IFFT) of a list of complex numbers.
    The length of the input must be a power of two.
    """
    n = len(a)
    if n == 0:
        return []
    if n & (n - 1):
        raise ValueError("Length of input must be a power of 2")
    if n == 1:
        return a

    even_part = ifft(a[::2])
    odd_part = ifft(a[1::2])

    angle = 2 * math.pi / n
    w_n = cmath.exp(1j * angle)
    w = 1 + 0j
    result = [0j] * n

    for k in range(n // 2):
        t = w * odd_part[k]
        result[k] = (even_part[k] + t) / 2
        result[k + n // 2] = (even_part[k] - t) / 2
        w *= w_n

    return result

# Example usage
data = [0+0j, 1+0j, 2+0j, 3+0j]
result = ifft(data)

• Because the routine divides by 2 at every recursion level, the overall scaling is 1/4, matching the mathematical IFFT factor 1/N for N = 4.

• Evaluating the inverse FFT formula
xₖ = (1/N) ∑_{n=0}^{N-1} Xₙ e^{j2πkn/N}
for the input [0, 1, 2, 3] gives

k = 0 : (0+1+2+3)/4 = 1.5  
k = 1 : (0 + 1·j − 2 − 3·j)/4 = −0.5 − 0.5 j  
k = 2 : (0 − 1 + 2 − 3)/4 = −0.5  
k = 3 : (0 − 1·j − 2 + 3·j)/4 = −0.5 + 0.5 j

which is exactly what the function returns.
